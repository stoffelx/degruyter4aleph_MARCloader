# csh-script for De Gruyter data loader
The current script triggers an automatic download of MARC-Data from De Gruyter according to the provider's specifications given at: https://oschmalfuss.github.io/degruyter/DG-title-records.html<br>It requires input from a CSV holdings list as available at DG Data portal, see: https://www.degruyter.com/publishing/services/for-librarians/tools-resources
